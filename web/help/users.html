admin
Ist das admin-Flag gesetzt, bekommt der User beim login die Maske mit den vollen Zugriffsmoeglichkeiten auf das prod-sys.

poster
Ist das poster-Flag gesetzt, hat der User die Moeglichkeit die eingehende Post zu bearbeiten. Das poster-Flag bezieht sich immer nur auf User, die keine admins sind.
